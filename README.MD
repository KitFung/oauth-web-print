# OAuth Web Print

## Prerequest
- python 2 or 3
- bower

## How to run

Use Python 2,

```
$ virtualenv env
$ source env/bin/activate
$ pip install -r requirement.txt
$ bower install
```

Use Python 3,

```
$ python3 -m venv env
$ source env/bin/activate
$ pip install wheel
$ pip install -r requirement.txt
$ bower install
```
